## Assign Plate IDs For Locations

**Description**:

HTTP GET request to assign plate IDs for locations

**Endpoint**:

    https://gws.gplates.org/reconstruct/assign_points_plate_ids

**Parameters**:

- ***model*** : name for the [reconstruction model](/models) (if not specified, will use the default model)
    
- ***with_valid_time***: if this flag is set, the server will return valid time

**Examples:** 

**&#128994; E1:** get plate IDs for locations (-10,50) (-130,-70) (0,0)

**Click this [link](https://gws.gplates.org/reconstruct/assign_points_plate_ids?lons=-10,-130,0&lats=50,-70,0) to try this example in a web browser.**

Alternatively, use curl command.

```shell
curl "https://gws.gplates.org/reconstruct/assign_points_plate_ids?lons=-10,-130,0&lats=50,-70,0"
```

**&#128994; E2:** get plate IDs for locations (-10,50) (-130,-70) (0,0) with valid time

**Click this [link](https://gws.gplates.org/reconstruct/assign_points_plate_ids?lons=-10,-130,0&lats=50,-70,0&with_valid_time) to try this example in a web browser.**

Alternatively, use curl command.

```shell
curl "https://gws.gplates.org/reconstruct/assign_points_plate_ids?lons=-10,-130,0&lats=50,-70,0&with_valid_time"
```
