You can use **wget** or **curl** to access the GPlates Web Service.

```shell
curl "https://gws.gplates.org/reconstruct/coastlines/?time=140"
```

or

```shell
wget -qO - "https://gws.gplates.org/reconstruct/coastlines/?time=140" 
```
