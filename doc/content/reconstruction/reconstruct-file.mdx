## Reconstruct File

**Endpoint:**

    https://gws.gplates.org/reconstruct/reconstruct_files

**Parameters:**

*time* : time for reconstruction [required]

*model* : name for reconstruction model [defaults to default model from web service settings]

**Examples:**

Check out the [online live demo](https://portal.gplates.org/service/reconstruct_file_page/#).

A Python example of using this service can be found [here](https://github.com/GPlates/gplates-web-service/blob/master/django/GWS/test/test_reconstruct_files.py).
