## Get All Plate IDs

**Description**:

HTTP GET request to get all the plate IDs in a reconstruction tree at a given time

**Endpoint**:

    https://gws.gplates.org/rotation/get_plate_ids

**Parameters**:

- ***time*** : time at which to get the plate IDs [required]

- ***model*** : the [reconstruction model](/models) name (if not specified, will use the [default model](/models#defaultmodel))

**Return**:

a list of plate IDs at the given time in the reconstruction tree

**Examples:** 

**&#128994; E1:** get all the plate IDs in the reconstruction tree at 100Ma

**Click this [link](https://gws.gplates.org/rotation/get_plate_ids?time=100) to try this example in a web browser.**

Alternatively, use curl command.

```shell
curl "https://gws.gplates.org/rotation/get_plate_ids?time=100"
```
