## Topological Plate Boundaries

**Description**:

HTTP GET request to retrieve reconstructed topological plate boundaries

**Endpoint**:

    https://gws.gplates.org/topology/plate_boundaries
    
**Parameters**:

- ***time*** : time for reconstruction [default=0]

- ***model*** : name for the [reconstruction model](/models) (if not specified, will use the default model)

**Return**:

JSON data containing reconstructed plate boundary features
